# Mercan Works Official Website

A high-performance, aesthetically driven landing page for Mercan Works, built with React, Tailwind CSS, and Framer Motion.

## Aesthetic Direction
**"Bioluminescent Intelligence"**
- A deep, dark void (`#02040F`) representing the unknown/potential.
- Living, breathing "Coral" (`#FF6B6B`) light representing organic growth and the company name.
- Sharp, crystalline data structures representing AI and analytics.

## Tech Stack
- **Framework:** React (Vite)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Project Structure
- `src/App.jsx`: Main single-page application logic.
- `src/index.css`: Global styles, fonts, and Tailwind directives.
- `tailwind.config.js`: Custom theme configuration (colors, animations).
